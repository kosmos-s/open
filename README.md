# 공부 목표 타이머 앱 (Study Goal Timer)

## 📌 프로젝트 소개
이 프로젝트는 **Jetpack Compose**를 사용하여 만든 간단한 **공부 목표 타이머 앱**입니다.  
사용자가 하루 목표 공부 시간을 설정하고, 총 공부 시간과 순공부 시간을 확인하며 공부를 관리할 수 있습니다.

---

## 📱 주요 기능

- **목표 시간 설정**: 사용자가 하루 공부 목표 시간을 분 단위로 설정할 수 있습니다.
- **총 공부 시간 표시**: 앱을 켜고 공부를 시작하면 총 공부 시간이 실시간으로 증가합니다.
- **순공부 시간 표시**: 집중 시작 버튼을 눌러 실제 집중한 시간을 기록할 수 있습니다.
- **진행률 표시**: 설정한 목표 시간 대비 현재 집중 시간을 그래프로 확인할 수 있습니다.
- **컨트롤 버튼**
  - **시작/일시정지**: 공부 타이머 시작 및 일시정지
  - **집중 시작/종료**: 집중한 시간만 카운트
  - **초기화**: 모든 기록 초기화

---

## ⚙️ 사용 방법

1. 앱을 실행합니다.
2. '일일 목표(분)' 입력란에 하루 공부 목표 시간을 입력하고 **목표 설정** 버튼을 누릅니다.
3. **시작** 버튼으로 공부를 시작합니다.
4. 집중 시간을 기록하려면 **집중 시작** 버튼을 눌러 순공부 시간을 측정합니다.
5. 필요 시 **일시정지** 또는 **초기화** 버튼을 눌러 기록을 관리합니다.
6. 공부 중인 시간과 진행률은 실시간으로 화면에 표시됩니다.

---

## 🛠️ 기술 스택

- **언어**: Kotlin  
- **UI**: Jetpack Compose  
- **아키텍처**: MVVM  
- **타이머**: Coroutine을 이용한 1초 단위 실시간 카운트

---

## 📌 참고 사항

- 현재 버전은 단순 타이머 기능만 포함하며, 과목별 타이머 및 앱 종료 후 상태 저장 기능은 구현되어 있지 않습니다.
- 추후 기능 확장 가능:
  - 과목별 타이머 관리
  - 앱 종료 후 상태 복원
  - 알림 및 소리 알람 추가
